The **WDC 65C816**, sometimes abbreviated as **65816**, is the main CPU used for the SNES, as part of its **S-CPU** chip.

More specifically, the SNES uses the **Ricoh 5A22**. This adds extra features to the processor, such as [[Multiplication]]/[[Division]] registers and [[DMA registers|DMA]] hardware.

See: **[[Main Page#65c816 guides|65c816 guides]]**
